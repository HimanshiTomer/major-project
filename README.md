# Urban Expansion Change Detection from Satellite Images

## Overview

This project focuses on detecting urban expansion by identifying changes between satellite images of the same geographical area captured at different time periods. The goal is to build a deep learning based computer vision system that can automatically highlight regions where significant land changes, especially urban growth, have occurred.

## Objective

To detect and analyze changes in urban areas by comparing satellite images taken at different time periods using computer vision and deep learning techniques.

## Problem Statement

Rapid urbanization leads to significant changes in land use and land cover. Manual monitoring of satellite imagery is time consuming and inefficient. This project aims to develop an automated change detection model that can compare multi temporal satellite images and detect areas of urban expansion accurately.

## Dataset

Primary Dataset: OSCD (Onera Satellite Change Detection Dataset)

* Satellite Source: Sentinel-2
* Data Type: Multi-temporal satellite image pairs
* Labels: Binary change masks (change vs no-change)
* Access: Public and free for research use
